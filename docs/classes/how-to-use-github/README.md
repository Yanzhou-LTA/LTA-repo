# 如何使用 `Github` ?

`Github` 是一个非常好用的平台(或者说网站), 大家或许早就听说过, 或者在上面下载过一些好用的软件, 但 `Github` 的用处远不止这么简单, 它是一个非常强大的代码托管平台, 也是程序员们进行协作开发的好帮手, 下面就来介绍一下 `Github` 的一些基本用法

## 注册账号

首先, 你需要注册一个 `Github` 账号, 访问 [Github 官网](https://github.com), 点击右上角的 `Sign up` 按钮, 然后按照提示填写你的用户名、邮箱和密码, 注册完成后你就可以登录 `Github` 了

## 浏览 `Github`

`Github` 上有成千上万的开源项目, 你可以通过搜索功能找到你感兴趣的项目, 也可以浏览热门项目和推荐项目, 每个项目都有一个专门的页面, 你可以在页面上查看项目的代码、文档、问题和贡献者等信息

有一些仓库汇集了很多有趣的项目, 例如:

- [HelloGitHub](https://github.com/521xueweihan/HelloGitHub)
- [GitHubDaily](https://github.com/GitHubDaily/GitHubDaily)

## 使用 `Github` 保存你的开发

运用 `Github` 你可以随时随地地保存和管理你的代码, 你可以创建一个新的仓库(Repository), 然后将你的代码上传到这个仓库中, 这样你就可以在任何地方访问和修改你的代码了

不仅仅是代码, 你还可以将文档、图片、视频等文件上传到 `Github`, 这样你就可以方便地管理和分享你的各种资源了

## 协作开发

`Github` 最强大的功能之一就是支持多人协作开发, 你可以邀请其他人加入你的仓库, 他们可以帮助你修改代码、修复问题和添加新功能, 你也可以参与其他人的项目, 提交你的修改建议和代码贡献

比如, 大家可以通过加入我们的 [LTA](https://github.com/Yanzhou-LTA) 组织, 一起参与到更多有趣的项目中来

如果需要帮助, 可以在 `Discussions` 版块发起讨论, 我们会尽力帮助你解决问题

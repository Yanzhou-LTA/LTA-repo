# 什么是 Github?

- 🌍 **代码托管平台**: 程序员们的"Facebook"
- 🤝 **协作开发**: 多人共同完成项目的利器
- 📦 **资源管理**: 不仅仅是代码，文档、设计稿皆可托管
- 🚀 **开源社区**: 全球最大的开源软件构建平台

<br>
<br>

> "Github 是一个非常好用的平台，也是程序员们进行协作开发的好帮手"

# 注册账号

开启你的开源之旅

1. 访问 [Github 官网](https://github.com)
2. 点击右上角的 `Sign up` 按钮
3. 填写用户名、邮箱和密码
4. 完成验证并登录

# 保存你的开发

### 创建仓库 (Repository)

- ☁️ **云端存储**: 随时随地访问代码
- 🔄 **版本控制**: 记录每一次修改
- 📂 **多类型支持**: 代码、文档、图片、视频

<br>

> 就像你的云端硬盘，但是专为开发者设计。

# 协作开发

多人力量大

- 👥 **邀请成员**: 邀请他人加入仓库
- 🔧 **共同维护**: 修改代码、修复 Bug、添加功能
- 🌟 **参与开源**: 向其他项目提交 Pull Request (PR)

<br>
